If there is any changes(pull request) in GitHub
1. Accept the pull request in GitHub.
2. $ git pull : run this command in local git terminal
3. Do you changes
4. Add the changes using # git add 
5. Commit the changes using # git commit
6. Push the changes to git hub using # git push 
7. Check the git hub if the changes are delivered

Between every steps run #git status and #git log commands to see the changes in steps.
